# Port-detection
Port-detection用于批量探测端口开放情况及版本信息  
## 运行环境  
> python 3.8  
## 功能：  
1. 结合了masscan的扫描速度和nmap的端口识别功能  
2. 结果保存在txt文件中  
## 用法：
```
python　Port_detection.py
已附带windows下使用的masscan，结合自身情况更改masscan路径
```
